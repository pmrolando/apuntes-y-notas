# Git

Inicializar repositorio

> `$ git init .` 

Clonar repositorio

> `$ git clone <url>` 


Configuraciones

> `$ git config --global user.name "Nombre"`

> `$ git config --global user.email micorreo@dominio.com`

> `$ git config --global --unset user.name`

> `$ git status`

> `$ git add .`

> `$ git log`
> `$ git checkout`
> `$ git branch`
> `$ git push`
> `$ git pull`
> `$ git merge`
